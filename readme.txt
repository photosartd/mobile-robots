Zum Ausführen des Beispielskripts "example.py" benötigen Sie eine funktionierende Python-Version >= 3.6 und
die Pakete "Kivy" und "numpy". Python 3.6 erfordert zusätzlich das Paket "importlib_resources".
Diese können Sie z. B. mit pip installieren: "pip3 install Kivy numpy" bzw. "pip3 install Kivy numpy importlib_resources".
Anschließend öffnen Sie in diesem Verzeichnis ein Terminal und starten die Beispieldatei mit "python3 example.py".
